Know-show table Example

| Step | Know                                                         | Reason                                                       |
| ---- | ------------------------------------------------------------ | ------------------------------------------------------------ |
| P    | $a,b,c\in \mathbb{Z}, a \ne 0, a|b, b|c$                     |                                                              |
| P1   | $s \in \mathbb{Z}$ and $t\in \mathbb{Z}$ exists for $a \times s = b, b \times t =c$ | $\because (\exist q \in \mathbb{Z}) (n = m \times q); \therefore n|m$ |
| P2   | $c = (a \times s) \times t$                                  | Substitution for b                                           |
| P3   | $c = a \times (s \times t)$                                  |                                                              |
| Q1   | $q \in \mathbb{Z}$ exists for $c=a \times q$                 | $\because (\exist q \in \mathbb{Z}) (n = m \times q); \therefore n|m$ |
| Q    | $a|c$                                                        |                                                              |
